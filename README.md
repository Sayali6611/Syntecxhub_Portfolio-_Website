# Portfolio Website

This is a responsive single-page portfolio website created using **HTML and CSS only** as part of the **Syntecxhub Internship Task**.

## Features
- Clean and modern UI
- Responsive design for mobile and desktop
- Fixed navbar with mobile menu toggle
- Skills and Contact section hover effects
- Simple and beginner-friendly code

## Technologies Used
- HTML5
- CSS3

##  Sections Included
- About
- Skills
- Projects
- Contact

##  How to Run
1. Download or clone this repository
2. Open `index.html` in your browser

##  Internship Task
This project was created and submitted as part of the **Syntecxhub Frontend Internship Program**.
